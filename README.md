
# 📋UAS PRAKTIKUM DASAR PEMROGRAMAN MOBILE




## Run Locally

Clone the project

```bash
  git clone https://github.com/Santoss104/UAS-LAB-DPM.git
```

Go to the project directory

```bash
  cd UAS-LAB-DPM
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
